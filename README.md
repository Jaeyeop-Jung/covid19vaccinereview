# 코로나 백신 리뷰

💡 서비스 소개

코로나 19 사태 중 백신 투여 후 다양한 부작용이 나타나는 상황입니다

해당 커뮤니티를 통해 사용자들의 부작용을 확인하고, 중증으로 나아가는 문제를 예방했으면 하는 바람으로 제작했습니다

각 백신 별 게시글 작성과 이미지 첨부 기능, 댓글 기능을 통해 자신의 경험을 공유할 수 있습니다!


🖥  API

- Spring Security와 JWT토큰을 활용한 로그인
- Google Oauth2를 활용한 로그인
- 게시글, 댓글 작성 및 조회
- 대댓글 작성
- 게시글에 이미지 첨부 
- 게시글, 댓글 좋아요 기능

👍  특징
- Front-End와 협업
- RestAssured를 활용한 통합 테스트코드 
- CustomException을 통해 손쉬운 예외 추적
- Swagger를 사용하여 API 명세
- Jasypt를 활용한 환경설정 암호화
- Github Action을 활용한 CI 구축
- CodeDeploy를 활용한 CD 구축
- Nginx를 활용한 무중단 배포 시스템 구축
